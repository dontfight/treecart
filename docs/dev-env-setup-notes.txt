=====================
X Debug
=====================

Already installed w/ Netbeans, if not follow the directions below

1. Get Xdebug that matches your PHP (the one installed with WAMP ... make sure it's thread safe )
2. Place XDebug dll in WAMP_path\php\extensions.
3. Add these lines to the php.ini under C:\wamp\Apache2\bin\php.ini

[XDebug]
zend_extension="C:\wamp\bin\php\php5.3.0\ext\php_xdebug-2.0.5-5.3-vc6.dll"
xdebug.remote_enable=on
xdebug.remote_host=localhost
xdebug.remote_port=9000
xdebug.remote_handler=dbgp
;xdebug.profiler_output_dir="C:\Temp"


4. !!even though it's included in the extensions directory, don't add to the extentions area like: extension=php_xdebug-2.0.5-5.3-vc6.dll
5.  phpinfo() sould inlcude an xdebug section and this text:
This program makes use of the Zend Scripting Language Engine:
Zend Engine v2.3.0, Copyright (c) 1998-2009 Zend Technologies
    with Xdebug v2.0.5, Copyright (c) 2002-2008, by Derick Rethans

7. In Wamp restart all services.
